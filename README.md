## 📁 Структура проекта

### 🔰 Базовые конструкции

| Файл | Описание | Пример использования |
|------|----------|----------------------|
| `01_int.py` | Ввод числа и преобразование в `int` | ```s = 25 -> print: 25\ninput: "100" -> print: 100``` |
| `02_range.py` | Генерация последовательности чисел с шагом | `range(55,99,3)` → вывод: 55, 58, 61... 97 |
| `03_len.py` | Подсчёт длины строки и списка | `len("hello")=5`, `len("Водопад")=7`, `len([10,20,30])=3` |
| `while.py` | Основы цикла `while` | Счёт от 1 до 5 и от 1 до 10 с форматированием |
| `while_task_01.py` | Обратный отсчёт | Цикл от -1 до -20: "Число -1", "Число -2"... |
| `while_task_02.py` | Чётные и нечётные через `while` | Чётные: 2,4...20; Нечётные: 1,3...21 |
| `while_task_03.py` | Выход из цикла по `0` | Ввод чисел до тех пор, пока не введён 0 |

### 🔄 Работа с типами данных

| Файл | Описание | Пример использования |
|------|----------|----------------------|
| `04_list.py` | Создание списков и преобразование типов | `["1","2","3"] → [1,2,3]`, `"hello" → ['h','e','l','l','o']` |
| `04_list_task_01.py` | Работа с типами `int` и `str` | Преобразование 5 и "5": сумма = 10, конкатенация = "55" |
| `04_list_task_02.py` | Сложение чисел и конкатенация строк | 23+34+12=69; "23"+"34"+"12"="233412" |
| `05_map.py` | Применение `map()` для преобразования типов | `[1.5,2.7,3.9,4.2,5.0] → [1,2,3,4,5]` |
| `06_split.py` | Разделение строк (IP-адрес, дата) | `"192.168.1.1" → ['192','168','1','1']` |

### 📊 Строковые методы

| Файл | Описание | Пример использования |
|------|----------|----------------------|
| `07_isdigit.py` | Проверка, состоит ли строка из цифр | `"23".isdigit()=True`, `"t".isdigit()=False` |
| `08_isalpha.py` | Проверка, состоит ли строка из букв | `"23".isalpha()=False`, `"t".isalpha()=True` |
| `09_isupper.py` | Проверка на верхний регистр | Ввод "HELLO" → `isupper()=True` |
| `10_islower.py` | Проверка на нижний регистр | Ввод "hello" → `islower()=True` |

### 📌 Работа со списками и индексами

| Файл | Описание | Пример использования |
